# Day 01: System Setup & The Execution Console 🛰️

Welcome to Day 1 of the 30 Days of R challenge! Today, we are setting up your development environment. R is the programming language engine, while RStudio is the Integrated Development Environment (IDE) interface we use to write and run code.

## 🛠️ Step-by-Step Installation Matrix

### Step 1: Install the R Engine
1. Go to the Comprehensive R Archive Network (CRAN): `https://cloud.r-project.org/`
2. Select your operating system (Windows, macOS, or Linux).
3. Download and run the installer for the base system.

### Step 2: Install RStudio (The Interface)
1. Go to Posit's official download portal: `https://posit.co/download/rstudio-desktop/`
2. Download the free desktop installer for your machine.
3. Install it using the default configurations.

## 🧠 Core Concept: The Interface Layout
When you launch RStudio, your workspace is split into four primary quadrants:
1. **Source Editor (Top Left):** Where you write your `.R` script files.
2. **Console (Bottom Left):** The live interactive brain where code runs instantly.
3. **Environment (Top Right):** Tracks every active matrix, dataset, and variable in memory.
4. **Files/Plots Viewer (Bottom Right):** Displays your charts, directory structures, and help files.
